## 2024-05-23 - Reading Progress Bar
**Learning:** Users on long-form content (blog posts) benefit significantly from visual feedback on their reading progress, especially on mobile where scrollbars might be hidden.
**Action:** Implement non-intrusive progress indicators for article-heavy pages to improve engagement and reduce bounce rates.

## 2024-05-23 - Implementation Safety
**Learning:** When applying UX enhancements to complex layouts, replacing the entire file can lead to accidental deletion of critical components (SEO, TOC, Images).
**Action:** Always use targeted diffs or careful reconstruction when modifying existing layout components. Always verifying existing file content before overwriting.
